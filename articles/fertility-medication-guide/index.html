不妊治療で使う薬一覧 | 用途・違い・注意事項まとめ

不妊治療や体外受精では、周期ごとにいくつかの薬を使い分けます。ここでは、薬の役割に加えて、ゴナールエフとフォリルモンのような似た薬の違いも迷いにくい形で整理しています。

すぐ見たい薬へ

ゴナールエフ フォリルモン レコベル オビドレル ジュリナ デュファストン

薬のグループを先に整理

排卵誘発剤

卵胞を育てる薬

採卵に向けて卵胞の発育を促します。注射薬ではゴナールエフ、フォリルモン、レコベル、HMG製剤などが代表です。

トリガー

採卵前の最終成熟を進める薬

オビドレル、HCG注射、スプレキュアなどがあり、採卵前の決められた時刻に使うのが重要です。

エストロゲン製剤

子宮内膜を整える薬

移植周期で使われやすく、ジュリナやエストラーナテープが代表です。内膜の厚さを整える役割があります。

プロゲステロン製剤

黄体補充をする薬

移植後の着床環境を支えるために使われ、デュファストン、ルトラール、ルティナス、ワンクリノンなどがあります。

排卵誘発剤

排卵誘発剤

ゴナールエフ

FSH製剤 / 自己注射で使われることが多い

卵胞を複数育てるために使う注射薬です。ペン型で自己注射しやすい案内になっていることが多く、毎日ほぼ同じ時間に続けます。

  • 向いている場面: 採卵周期で卵胞発育をコントロールしたいとき
  • 注意点: お腹の張りや急な体重増加があれば卵巣が反応しすぎていないか確認が必要です
フォリルモンとの違い: どちらもFSH製剤として使われますが、案内される製剤の形やクリニックでの採用状況が異なります。役割はかなり近く、「卵胞を育てる主役の注射」という理解で大丈夫です。
排卵誘発剤

フォリルモン

FSH製剤 / 卵胞発育を促す注射薬

フォリルモンも卵胞を育てるためのFSH系の注射薬です。ゴナールエフと同じく、採卵数を確保したい周期で使われることがあります。

  • 向いている場面: ゴナールエフと同様に卵胞発育を促したいとき
  • 注意点: 注射部位の痛み、張り感、卵巣過剰刺激のサインに注意します
ゴナールエフとの違い: 働きの方向性はかなり似ています。実際の使い分けは、施設の方針、費用感、在庫、注射の扱いやすさなどで決まることが多いです。
排卵誘発剤

レコベル

FSH製剤 / 体重やAMHをもとに量を調整しやすい

レコベルは、体重やAMHを参考に投与量を考えることがある排卵誘発注射です。卵巣の反応を見ながら使われます。

排卵誘発剤

HMG製剤

FSH + LH 活性 / 卵胞発育を後押し

HMG製剤はFSHに加えてLH活性を含み、反応を見ながら使い分けられることがあります。FSH単独製剤と方針が異なるケースがあります。

排卵誘発剤

クロミッド

内服薬 / 軽い刺激で使われることがある

脳からのホルモン分泌を促して排卵を助ける内服薬です。注射ほど強くない刺激で始める周期に使われることがあります。

排卵誘発剤

フェマーラ

レトロゾール製剤 / 内膜とのバランスを見て使うことがある

クロミッドと並んで使われることがある内服の排卵誘発薬です。クリニックによって選び方が変わります。

トリガー注射・点鼻薬

トリガー

オビドレル

hCG製剤 / 自己注射で使われることが多い

採卵前の最終成熟を進めるための自己注射です。採卵時刻から逆算して打つため、時間厳守がとても大切です。

トリガー

HCG注射

トリガー注射 / 採卵前の仕上げ

卵子の最終成熟を促すために使います。オビドレルも広い意味では同じ役割の仲間で、製剤の形や使い方に違いがあります。

トリガー

スプレキュア

点鼻薬 / 指示された回数と時刻が大切

鼻に使うタイプのトリガー薬です。自己注射ではなく点鼻で案内されることがあり、鼻炎が強いときは相談が必要な場合があります。

エストロゲン製剤

エストロゲン製剤

ジュリナ

内服薬 / 子宮内膜を整える

移植周期で内膜を厚くしたいときに使われる内服薬です。飲み忘れが続くと内膜の育ちに影響することがあります。

エストロゲン製剤

エストラーナテープ

貼り薬 / 貼り替えスケジュール管理が大切

皮膚からエストロゲンを補う薬です。貼り替え忘れやかぶれに注意します。

エストロゲン製剤

内膜調整のホルモン薬

総称ページ

ジュリナやエストラーナテープのように、移植に向けて子宮内膜の厚さやタイミングを整える薬の総称です。

プロゲステロン製剤

プロゲステロン製剤

デュファストン

内服薬 / 黄体補充

移植後の黄体ホルモン補充に使われる内服薬です。開始日が大切なので、指示通りに始めます。

プロゲステロン製剤

ルトラール

内服薬 / 眠気やだるさに注意

黄体補充の代表的な薬のひとつです。判定日前後まで継続することが多く、自己判断で中止しないことが大切です。

プロゲステロン製剤

ルティナス

腟錠 / 局所で補充

腟剤として黄体補充を行う薬です。使用後に外へ出てくるように見えても、一定量は吸収されています。

プロゲステロン製剤

ワンクリノン

ゲル状の腟剤 / 残渣が出ることがある

ゲル状の黄体補充薬で、白い塊のような残りが出ることがあります。自己判断で洗い流したり中止したりしないようにします。

受診を急ぎたい症状

強い腹痛、息苦しさ、急な体重増加、片脚の腫れ、激しい頭痛、発熱、強い出血などがあるときは、自己判断で様子を見すぎず通院先へ連絡してください。

薬の予定も含めて流れを確認する

生理開始日や採卵日、移植日を入れると、治療の流れを日付ベースで確認できます。

スケジュールを確認する

どの薬がどの時期に出やすいかも合わせて見やすくなります。

この記事は一般的な説明をまとめたものです。服薬や治療方針は、必ず通院先の案内をご確認ください。
